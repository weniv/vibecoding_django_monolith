나는 Django를 이제 막 배우는 초보 개발자야. 강사님이 `명령어_백업.md` 파일을 이전시간에 작성하여 주었어.

이를 참고하여 아래 URL 설계를 바탕으로 Django 간단한 웹 애플리케이션을 구현하는 스탭을 `명령어.md` 파일로 작성할 예정이야. `명령어.md` 파일을 만들고, 파일에 들어갈 내용을 작성해줘. `(추가)`라고 표시된 부분은 내가 추가로 원하는 URL이야.

```
'/' : 홈 페이지
'/about': 소개 페이지
'/contact': 연락처 페이지
'/blog': 블로그 메인 페이지
'/blog/1': 블로그 1번 글 페이지
'/blog/2': 블로그 2번 글 페이지
(추가) '/blog/태그명': 특정 태그의 블로그 글 페이지
'/admin': 관리자 페이지
```
