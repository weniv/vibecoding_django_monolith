이것은 Django CBV(Class Based View)와 Bootstrap으로 만든 블로그입니다.

주요 기능

다음과 같은 기능을 제공합니다:

  • 회원가입 및 로그인/로그아웃
  • 게시글 작성, 수정, 삭제
  • 게시글 목록 조회 및 상세 보기
  • 댓글 작성, 수정, 삭제
  • 태그 기능
  • 마크다운 지원
  • 조회수 및 좋아요 기능
  • 이미지 업로드 (프로필, 썸네일)

기술 스택

  • Backend: Django 5.x
  • Frontend: Bootstrap 5, Font Awesome
  • Database: SQLite (개발용)
  • 기타: Markdown 지원, Pillow (이미지 처리)

이 프로젝트는 Django 학습 목적으로 제작되었습니다.
