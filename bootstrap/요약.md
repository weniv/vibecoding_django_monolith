# Bootstrap

* Bootstrap은 오픈 소스 프론트엔드 라이브러리로, 반응형 웹 디자인과 모바일 우선 접근 방식을 지원합니다. HTML, CSS, JavaScript로 구성되어 있으며, 다양한 UI 컴포넌트와 그리드 시스템을 제공합니다. 
* Front-end 개발자들은 잘 사용하지 않습니다. UI가 이미 구현이 다 되어 있기 때문에 복잡한 커스터마이징이 어렵기 때문입니다. 그리고 Bootstrap으로 만든 사이트들은 다 비슷비슷하게 생겨서 차별화가 어렵습니다.
* 어떨때 사용하는지?
    1. 프론트엔드 개발자가 없을 때
    2. 빠른 프로토타이핑이 필요할 때
    3. 무료로 사용할 수 있는 UI 컴포넌트가 필요할 때(돈은 없고, 시간은 많을 때) - 부트스트랩 무료 템플릿이라고 검색
    4. 유료로 빠르게 서비스를 만들고 싶을 때(돈은 있는데, 시간이 없을 때) - 부트스트랩 유료 템플릿이라고 검색
