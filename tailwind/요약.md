# TailwindCSS

* 테일윈드는 서비스를 A to Z까지 빠르게 개발할 수 있도록 도와주는 도구입니다. 여러분들이 자주 사용하는 모든 CSS가 클래스로 이미 구현이 되어 있습니다.
* 그렇다고 CSS 파일이 모두 있는 것은 아니고 JS로 컨트롤 됩니다. 가볍고, 커스터마이징이 자유롭습니다.
* Front-end 개발자들이 가장 선호하는 CSS 프레임워크입니다. 이유는 커스터마이징이 자유롭고, 클래스 네이밍이 직관적이기 때문입니다.
* 허들이 있습니다! Tailwind CSS 속성을 모두 외워야 합니다.
* 프롬프트: Django 웹 페이지 다 개발하고 나중에 아래와 같이 명령하면 좋습니다. (처음부터 이렇게 명령하면 코드가 너무 길어져서 실패할 확률이 높습니다.)

```
지금 Django 프로젝트가 순수한 html, css, js로 되어 있는데 이것을 TailwindCSS로 바꿔줘. 바꿀 때에는 '애플 서비스'처럼 모던하고 심플한 디자인으로 바꿔줘. TailwindCSS 속성들을 적극 활용해줘.
```

```
지금 Django 프로젝트가 순수한 html, css, js로 되어 있는데 이것을 TailwindCSS로 바꿔줘. 바꿀 때에는 '디자인_시스템.md' 파일을 참고하여 이대로 디자인해줘. 이 디자인 파일에는 컬러 팔레트, 폰트 스타일, 버튼 스타일 등이 정의되어 있어. TailwindCSS 속성들을 적극 활용해줘.
```
