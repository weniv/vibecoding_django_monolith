# Django 블로그 마크다운 렌더링 후속 작업

게시물 내용을 마크다운으로 작성하고 HTML로 렌더링하기 위한 설정 가이드입니다.

## 1. Markdown 라이브러리 설치

```bash
pip install markdown
```

### 3-3. markdown_extras.py 파일 작성
```python
# main/templatetags/markdown_extras.py
import markdown
from django import template
from django.utils.safestring import mark_safe

register = template.Library()

@register.filter(name='markdown')
def markdown_format(text):
    """
    마크다운 텍스트를 HTML로 변환하는 필터
    """
    return mark_safe(markdown.markdown(text, extensions=[
        'markdown.extensions.fenced_code',  # 코드 블록 지원
        'markdown.extensions.tables',       # 테이블 지원
        'markdown.extensions.nl2br',        # 줄바꿈 지원
        'markdown.extensions.codehilite',   # 코드 하이라이팅
    ]))
```

## 7. 추가 마크다운 확장 기능 (선택사항)

더 많은 기능이 필요한 경우 추가 확장 기능을 설치할 수 있습니다:

```bash
pip install pymdown-extensions
```

markdown_extras.py에 확장 기능 추가:
```python
@register.filter(name='markdown')
def markdown_format(text):
    return mark_safe(markdown.markdown(text, extensions=[
        'markdown.extensions.fenced_code',
        'markdown.extensions.tables',
        'markdown.extensions.nl2br',
        'markdown.extensions.codehilite',
        'markdown.extensions.toc',          # 목차 생성
        'markdown.extensions.sane_lists',   # 리스트 개선
        'markdown.extensions.smarty',       # 스마트 타이포그래피
    ]))
```

## 8. 코드 하이라이팅 개선 (선택사항)

Pygments를 사용하여 코드 하이라이팅을 개선할 수 있습니다:

```bash
pip install Pygments
```

CSS 파일 생성:
```bash
python -c "from pygments.formatters import HtmlFormatter; print(HtmlFormatter(style='monokai').get_style_defs('.codehilite'))" > main/static/main/pygments.css
```

템플릿에 CSS 파일 링크 추가:
```html
<link rel="stylesheet" href="{% static 'main/pygments.css' %}">
```

## 참고사항

- `mark_safe()`: Django의 자동 이스케이프를 우회하여 HTML을 안전하게 렌더링합니다.
- 신뢰할 수 없는 사용자 입력에는 추가 보안 처리(HTML 새니타이징)가 필요할 수 있습니다.
- XSS 공격 방지를 위해 `bleach` 라이브러리 사용을 권장합니다:
  ```bash
  pip install bleach
  ```
